La barra de Navegación está lista para navegar a: 
Inicio, Usuarios, Admin
Tiene validaciones:
Campos vacíos.
Cantidad de caracteres.
Que al ingresar un NSS verifique que no exista uno igual.
Sólo el admin (si está dado de alta en el sistema) puede agregar y/o modificar los datos de los usuarios.

El (los) admin(s) puede(n) modificar sus datos si así lo desea, pero no puede alterar que sea de tipo de usuario admin,
en todo caso tiene que darse da baja. 
No puede darse de baja el admin si no existe otro admin dado de alta en el sistema 
(tiene que haber min 2 admins para que uno de ellos se pueda dar de baja).

Para su correcto funcionamiento y estilo, necesita de las carpetas subidas anteriormente.
Nota: los "href" de los botones de la barra de inicio depende de donde o en que carpeta dentro de su wampp server guarden los archivos
